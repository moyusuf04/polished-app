# Project Brief (Persistent)
- **Product vision:** Conversational breadth for the ambitious through bite-sized cultural lessons.
- **Coding conventions:** Strict TypeScript (no `any`), modular architecture, routes handle request/response only.
- **Quality gates:** Pre-commit hooks for lint/format, unit tests for core utils, UI responsive checks.
- **Key commands:** `npm run dev` (start), `npm test` (test), `npm run lint` (lint).
- **Update cadence:** Refresh this brief when introducing new major architectural patterns or team members.
